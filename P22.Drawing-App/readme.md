A simple and interactive Drawing App built with HTML5 Canvas, CSS, and JavaScript. Users can draw freely on the canvas using the mouse, adjust the brush size, choose different colors, and clear the canvas with a single click. This project demonstrates the use of the Canvas API, DOM manipulation, event handling, and real-time user interaction.

Features
Freehand drawing using the HTML5 Canvas API
Adjustable brush size (increase/decrease)
Custom color selection with a color picker
One-click canvas clearing
Responsive and clean user interface
Smooth drawing experience using mouse events
